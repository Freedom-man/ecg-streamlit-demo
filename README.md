# ECG Streamlit Demo

Локальное приложение для предварительного анализа 12-канальных ЭКГ моделями
FCN-1D и ResNet-1D, обученными на PTB-XL.

## Запуск

```powershell
pip install -r requirements.txt
streamlit run app.py
```

## Публикация

Для публикации на Streamlit Community Cloud корнем репозитория должна быть эта
папка. Основной файл приложения: `app.py`.

Приложение принимает демонстрационные записи PTB-XL или файл NumPy с формой
`(число отсчетов, 12)`. Частота демонстрационных записей составляет 100 Гц.

Интерфейс адаптирован для широких экранов, планшетов и мобильных устройств.
На узком экране боковая панель автоматически сворачивается и открывается
кнопкой в верхней части страницы.
